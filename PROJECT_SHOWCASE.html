K-Drama Actress Tier List

A social tier ranking platform for Korean drama actresses with AI recommendations, community leaderboards, and real-time collaboration.

Live Demo GitHub Repo
At a Glance
31 API Endpoints

132 Unit Tests

Built as a personal project to explore full-stack development at scale. The app lets users drag-and-drop rank Korean drama actresses into tiers (S+ through D), track dramas, share lists publicly, follow other users, and discover trending actresses through a community leaderboard. Features include AI-powered drama recommendations via Claude API, TMDB integration for actress and drama data, and a PWA-enabled frontend.

Features

Core Functionality

☯
Drag-and-Drop Tier Ranking
Rank actresses into S+, S, A, B, C, D tiers with intuitive drag-and-drop. 10-point weighted scoring system. Guest mode with local interaction.
🔍
TMDB Integration
Search and add any Korean actress from TMDB. Auto-fetches drama credits, gallery photos from 5 sources, and biographical data.
🏆
Community Leaderboard
Global rankings aggregated across all public users. Sort by score, lists, or top-tier count. Genre filtering. TTL-cached with MongoDB.
🔥
Trending Analytics
Trending tab showing actresses ranked by popularity x quality score. Surfaces rising favorites across the community.
👥
Social Follow System
Follow/unfollow users, view following list with ranked counts, follower/following stats. Follow button on shared tier lists.
🔗
Shareable Tier Lists
Public/link-only/private visibility. Custom share slugs. Dynamic OG meta tags for social previews. Screenshot capture for sharing.
⚖
Side-by-Side Comparison
Compare two users' tier lists with agreement stats, exact match percentage, and highlighted matching rows.
🤖
AI Recommendations
Claude-powered chat assistant that analyzes your rankings and watch history to suggest new dramas. Real-time SSE streaming.
📺
Drama Tracking
Rate dramas 1-10, track watch status (watching, watched, plan to watch, dropped). Full watchlist management page.
📊
Statistics Dashboard
Personal analytics: tier distribution, genre breakdown, total ranked/unranked counts. Per-actress community stats with tier distribution bars.

Database

MongoDB Collections

1
actresses
Core actress data: name, image, genre, dramas, gallery photos, birth info. Indexed on dramas.title.
2
user_rankings
Per-user tier placements. Compound unique index on (userId, actressId).
3
user_drama_status
Watch status and ratings per drama per user. Compound unique index on (userId, actressId, dramaTitle).
4
user_actresses
Which actresses each user has in their pool. Enables per-user customization beyond the default 36.
5
user_profiles
Display name, bio, share slug, visibility (private/link_only/public), picture. Unique indexes on userId and shareSlug.
6
user_follows
Social follow relationships. Compound unique index on (followerId, followingId). Indexed both directions for fast lookups.
7
leaderboard_cache
Cached leaderboard aggregation results. TTL index on cachedAt (5-minute expiry) for automatic cache invalidation.
Development Process

Built Across 5 Sprints

1
User Profiles Foundation
User profiles collection, profile API endpoints, UserSettings page with display name/bio/slug editor, SharedTierList page for viewing others' rankings with dynamic OG meta tags.
2
Share Tier Lists
Enhanced share modal with screenshot capture and link sharing. Auto-visibility upgrade prompt (private to link_only). Privacy controls with three visibility levels. Comprehensive UI tests.
3
Global Leaderboard
Aggregated rankings across public users with weighted 10-point scoring. Top-3 podium with gradient cards. Sortable by score, lists, or top-tier count. Genre filtering. TTL-cached in MongoDB.
4
Compare Lists & Community Stats
Side-by-side tier list comparison with agreement percentage and exact match stats. Per-actress community ranking cards with tier distribution bars, average scores, and rank badges.
5
Follow System & Trending
Follow/unfollow users with follower counts. Following page with user cards and compare shortcuts. Trending leaderboard tab with popularity x quality scoring algorithm.
